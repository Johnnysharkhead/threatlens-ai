AI驱动攻击 2026-03-23 复杂度: 高

Kimsuky利用ChatGPT伪造韩国军事ID证件：AI深度伪造武器化用于间谍鱼叉式钓鱼

朝鲜国家黑客组织Kimsuky使用ChatGPT生成伪造的韩国国防机构军事ID证件图片，绕过内容安全护栏，在针对军事人员、研究人员及人权活动人士的鱼叉式网络钓鱼中大幅提升邮件可信度。

攻击概况

攻击方
Kimsuky（朝鲜国家支持的高级持续性威胁组织，隶属于朝鲜侦察总局，长期从事针对韩国政府、学术界及军事目标的网络间谍活动）
攻击目标
主要目标包括：韩国军事相关人员、朝鲜问题研究人员、朝鲜人权活动人士、新闻记者；攻击方以伪装成韩国国防机构的官方名义实施定向鱼叉式钓鱼
经济损失
N/A（本次攻击以情报窃取和远程控制为目的，尚无公开的直接经济损失数据）
事件状态
持续中
事件时间线
2025年6月：Kimsuky发动ClickFix系列钓鱼活动（伪造CAPTCHA窗口），使用相同恶意软件基础设施；2025年7月17日：Genians安全中心（GSC）检测并披露本次AI伪造军事ID钓鱼活动；攻击活动性质为持续间谍行动
行业影响
本次事件标志着国家级APT组织已将生成式AI武器化为标准化间谍工具，开创了'AI辅助证件伪造+多阶段恶意软件'的复合攻击新范式。其深远影响体现在：第一，鱼叉式钓鱼攻击的可信度上限被AI技术大幅拉高，传统依赖人工识别钓鱼邮件的防御思路正在失效；第二，朝鲜黑客将AI滥用与其全球IT工人证件欺诈行动融合，显示出系统性的AI攻击能力建设；第三，所有拥有ChatGPT访问权限的攻击者理论上都可复制类似攻击，使高质量社会工程学攻击的准入门槛全面降低；第四，对AI平台提出了更高的意图识别和下游危害预防要求，现有基于语义的内容审核机制面临系统性挑战，需要引入基于行为意图推断的新一代内容安全机制。军事、情报和国家安全领域需将AI生成内容检测能力纳入核心防御体系。

攻击手法

攻击者以韩国国防机构名义发送鱼叉式钓鱼邮件，邮件主题伪装为'军事人员工作证样稿审核请求'。邮件附带由ChatGPT生成的高度仿真韩国军事ID证件PNG图片，以提升邮件可信度。绕过ChatGPT安全护栏的核心手法为提示注入（Prompt Injection）：攻击者不直接要求生成军事证件，而是将请求重新框定为'制作合法用途的样品设计'或'模板展示'，利用ChatGPT对'合法设计辅助'场景的豁免逻辑规避内容审核。邮件中同时附带恶意.zip压缩包，包含.LNK快捷方式文件，触发时通过混淆的PowerShell命令下载第二阶段载荷：伪装成Hancom Office更新的批处理文件（LhUdPC3G.bat），执行后实施内部数据窃取并建立远程控制通道。持久化机制通过Windows计划任务（Task Scheduler）实现。通信混淆使用Vigenere密码变体和字符串切片技术，并借助AutoIt脚本增强免杀能力，C2服务器为jiwooeng.co[.]kr。
LLM01 - Prompt Injection（利用提示注入绕过ChatGPT内容安全护栏，诱导AI生成被政策禁止的军事身份证件图像） ChatGPT（OpenAI生成式AI，用于生成伪造军事ID证件图片）.LNK快捷方式文件（恶意载荷初始投递）PowerShell（混淆命令执行）批处理脚本 LhUdPC3G.bat（第二阶段载荷，执行数据窃取和远程控制）AutoIt脚本（免杀增强）Windows Task Scheduler（持久化）Vigenere密码变体（C2通信混淆）TruthScan（防御方检测工具，用于验证AI生成图像，检测率98%）
利用的脆弱性
  • ChatGPT内容安全护栏绕过：通过提示注入重新框定请求语境，将生成军事证件的请求伪装为'合法用途的样品设计或模板制作'，规避AI对非法伪造政府证件的拒绝机制
  • 生成式AI图像内容审核机制不完善：当提示语义被刻意模糊化后，AI系统无法可靠识别生成内容的最终滥用意图
  • 深度伪造图像真实性检测滞后：生成的PNG图像经TruthScan检测被AI生成概率达98%，但普通收件人及邮件安全网关缺乏同等检测能力

攻击链分析

Kimsuky将生成式AI武器化，分两个并行维度展开攻击：一是利用ChatGPT提示注入生成伪造军事证件图片作为社会工程学诱饵，大幅提升钓鱼邮件可信度；二是通过恶意压缩包投递多阶段恶意软件，最终实现数据窃取与持久化远程控制。攻击链充分体现了国家级APT组织将AI能力与传统多阶段恶意软件技战术深度融合的新型作战模式。

启示与防御

AI在攻击中的角色
AI在本次攻击中扮演了'高级伪造文件制造工厂'的角色，核心价值体现在三个维度：一是能力替代——替代了专业证件伪造所需的图形设计专业知识、目标机构内部资料获取能力和大量人工制作时间，将原本需要专业团队完成的任务压缩为数分钟的提示工程；二是可信度倍增——AI生成的军事ID证件比传统PS伪造具有更高的视觉一致性和细节真实感，根本性提升了鱼叉式钓鱼邮件的可信度；三是攻击门槛压低——使本需高级技能的证件伪造能力下沉至一般技术水平的攻击者，预示着国家级APT战术的扩散风险。
AI特有脆弱性
  • ChatGPT提示注入漏洞：内容安全护栏可通过语义重构被系统性绕过，当请求被框定为'合法设计/样品制作'时，模型无法可靠判断生成内容的下游滥用意图
  • 生成式AI意图识别盲区：当前大语言模型的内容审核机制基于表层语义匹配，无法有效识别经过精心包装的恶意用途请求，存在系统性的语义欺骗漏洞
  • AI图像生成与真实证件的视觉可信度趋同：生成式AI已能产出足以欺骗人类感知（但可被算法98%概率识别）的政府证件级别伪造图像，人机感知差距被恶意利用
  • AI降低了国家级攻击能力的门槛：伪造证件过去需要专业图形设计师和实物情报，现在ChatGPT可在数分钟内生成高质量样本，将高技术成本攻击能力普及化
